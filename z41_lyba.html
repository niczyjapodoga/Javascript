Pętle w JavaScript

Pętla for

for(inicjalizacja; warunek; zmiana)
{
kod
}

Pętla wykonuje kod określoną liczbę razy.

Pętla while

while(warunek)
{
kod
}

Najpierw sprawdza warunek, potem wykonuje kod.

Pętla do...while

do
{
kod
}
while(warunek);

Wykona kod przynajmniej raz zanim sprawdzi warunek.
